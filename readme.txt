== Description ==

Adminpress is a very simple plugin that will help to customize the WordPress admin panel. You will have lots of options and those are very easy to use. The plugin is very user friendly.

**Features**<br>
1. Common Styles<br>
2. Menu Styles<br>
3. Login page styles<br>
4. Menu Management<br>
5. Dashboard Widget<br>
6. Custom Admin Bar<br>
7. Footer Text<br>
8. Replace text<br>
9. Ajax Search<br>
10. Custom css and js<br>


== Installation ==

**Way 1:**<br>
1. Download the plugin file<br>
2. Unzip the file into a folder on your hard drive<br>
3. Upload via FTP into your plugins (/wp-content/plugins/) folder<br>
4. Activate the plugin from your dashboard<br>

**Way 2:**<br>
1. From Dashboard > Plugins > Add New > Upload the plugin<br>
2. Activate the plugin<br>
